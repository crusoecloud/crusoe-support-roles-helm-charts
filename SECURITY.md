# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it
responsibly by emailing [support@crusoecloud.com](mailto:support@crusoecloud.com).

Please include as much detail as possible to help us understand and reproduce
the issue, such as:

- A description of the vulnerability and its potential impact
- Steps to reproduce
- Affected versions or configurations
- Any suggested remediation

Please do **not** disclose the vulnerability publicly until we have had a chance
to investigate and address it.

## Response Timeline

We aim to acknowledge receipt of your report within **48 hours**. We will keep
you informed of our progress as we investigate and work toward a resolution.

Thank you for helping keep this project and its users safe.
